CS 280: Web Interface Development

Fall 2020

Learning to work with the tools and technologies that are used to build and create web sites is both empowering and fun. With these tools, you can build your own web sites and customize the experience of those you visit. You can also create interactive applications for storing and manipulating personal or enterprise data. And you can dynamically generate images and graphics to help visualize data sets.

The goal of this course is to explore these technologies - HTML, CSS, and Javascript - that define the structure, presentation, and interactions for modern web development. In addition to covering the basics of how to use these tools, we will explore guidelines to ensure the sites we create are accessible for a variety of users, including users with disabilities. After completing this course, you will have a solid foundation to continue learning more advanced web development techniques.

Throughout this course, you will be provided with opportunities - both in class and in weekly lab assignments - to practice each of these skills independently of the others. In addition, you will design and build a prototype application based on your own personal interests. The goal of this project-based course structure is for you to finish the semester with a rudimentary application that you can extend as part of a professional portfolio.

Detailed Course Objectives

Following the successful completion of this course, students will be able to: * Summarize the key steps and processes for retrieving and displaying a web page. * Structure text documents as HTML and publish them on a server. * Apply accessibility standards to web documents. * Create and adapt CSS style sheets for consistent web site presentation and styling. * Use a web framework to create a modern, responsive web site. * Generate HTML content dynamically with Javascript. * Create handlers to detect and respond to user input events. * Store and retrieve data in client-side storage. * Build images dynamically using provided data. * Learn to consult a web framework documentation and tutorials for additional help.
